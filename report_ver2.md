# [서비스 가이드] 생성형 AI 기반 광고 콘텐츠 제작 및 업로드 자동화

이 문서는 프로젝트의 전체 흐름, 주요 기능 및 내부 로직을 팀원 공유용으로 정리한 기술 문서입니다.

---

## 1. 서비스 개요
- **정의**: 소상공인이 상품 정보와 홍보 목적을 입력하면, AI가 최적의 광고 문구와 이미지(피드/스토리용)를 제작하고 인스타그램에 바로 게시해 주는 자동화 도구입니다.
- **입력**: 상품명, 상세 설명, 홍보 목적(할인, 신상 등), 광고 스타일, 원본 상품 사진
- **결과**: 인스타그램 피드용 문구/이미지, 스토리용 카피/이미지, 실제 인스타그램 포스팅

---

## 2. 전체 흐름 (End-to-End Flow)
1.  **정보 입력**: 사용자가 상품의 특징과 광고의 목적(예: 신상품 홍보)을 선택합니다.
2.  **전략 및 문구 생성**: 입력된 목적에 맞춰 AI가 타겟팅된 광고 카피와 인스타 스토리용 짧은 문구를 생성합니다.
3.  **광고 이미지 제작**: AI가 상품의 특징을 살린 1:1 이미지를 생성합니다.
4.  **스토리 레이아웃 합성**: 생성된 이미지 위에 선택한 스토리 문구를 자동으로 디자인하여 합성합니다.
5.  **인스타그램 전송**: 최종 완성된 이미지와 문구를 클릭 한 번으로 인스타그램에 업로드합니다.

---

## 3. 구성 요소 설명
- **Frontend (Streamlit)**: 인터랙티브한 웹 화면 구성 및 사용자 입력 관리
- **Service Layer (Python Services)**: 텍스트 생성, 이미지 생성, 업로드 로직 등 핵심 비즈니스 로직 처리
- **AI Models**:
    - **OpenAI (GPT)**: 광고 문구 전략 수립 및 이미지 생성용 프롬프트 최적화
    - **Hugging Face (FLUX)**: 고품질 광고 이미지 생성
- **Image Engine (Pillow)**: 이미지 규격 변경 및 텍스트 오버레이 합성
- **SNS API (Meta Graph API)**: 인스타그램 자동 포스팅 처리

---

## 4. 주요 기능
- **마케팅 목적별 문구 생성**: 단순 생성이 아닌 '할인', '소개' 등 목적에 따른 맞춤형 카피라이팅
- **참조 이미지 기반 생성**: 업로드한 사진의 느낌을 반영한 고퀄리티 광고 이미지 제작
- **인스타 스토리 최적화**: 이미지 생성 및 가독성을 고려한 텍스트 자동 합성
- **기록 관리 (History)**: 과거에 생성했던 문구와 이미지를 조회하고 재사용 가능
- **원클릭 배포**: 별도의 다운로드 없이 외부 API를 통해 인스타그램에 즉시 게시

---

## 5. 핵심 로직 흐름
1.  **Prompt Building**: 사용자 입력을 AI가 이해하기 쉬운 상세 지시문으로 변환 (`utils/prompt_builder.py`)
2.  **Content Generation**: GPT와 FLUX 모델을 순차적으로 호출하여 텍스트와 이미지 에셋 확보 (`services/text_service.py`, `services/image_service.py`)
3.  **Visual Composition**: Pillow를 통해 이미지 위에 카피를 배치하여 최종 스토리 이미지 완성
4.  **Publication Pipeline**: 이미지를 전용 서버에 임시 호스팅한 후, Meta API 컨테이너를 통해 인스타그램 게시 (`services/instagram_service.py`)

---

## 6. 코드 구조 간단 설명
- **`/services`**: 광고 문구, 이미지, 인스타그램 연동 등 핵심 로직이 담긴 서비스 폴더
- **`/utils`**: AI에게 전달할 프롬프트를 전문적으로 조립하는 엔진 폴더 (`prompt_builder.py`)
- **`/schemas`**: 데이터의 주고받는 규격을 정의한 Pydantic 모델 폴더
- **`/models`**: 생성 내역을 저장하기 위한 데이터베이스 테이블 정의 폴더
- **`app.py`**: 화면을 그리고 사용자 요청을 서비스 레이어로 전달하는 메인 컨트롤러

---

## 7. 실행 및 사용 방법
- **실행**: 터미널에서 `streamlit run app.py` 입력
- **사용 예시**:
    - 상품명에 "수제 쿠키" 입력
    - 홍보 목적에 "매장 소개" 선택
    - [생성] 버튼 클릭 후 결과 확인
    - 마음에 드는 카피 선택 후 [인스타그램에 올리기] 클릭
