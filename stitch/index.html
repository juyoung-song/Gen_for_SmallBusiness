모바일 베이커리 PWA 시안 미리보기

온보딩 3단계와 메인 생성 화면 1종을 바로 확인할 수 있게 정리했습니다. 각 카드를 누르면 실제 HTML 화면으로 이동합니다.

STEP 01

브랜드 기본 정보

브랜드 이름, 메인 컬러, 분위기, 로고 업로드를 입력하는 시작 화면
STEP 02

스타일 레퍼런스

인스타 링크와 베이커리 레퍼런스 이미지를 등록하는 페이지
STEP 03

우리 가게 소개

선택 입력으로 매장 분위기와 브랜드 설명을 적는 마지막 온보딩 화면
CREATE

메인 생성 화면

기본 입력으로 시작하고, 상세 옵션 토글을 눌러 목적·타입·스타일을 펼칠 수 있는 통합 화면
