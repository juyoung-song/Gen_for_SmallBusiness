# 📷 인스타그램 OAuth 연동 팀원 가이드

이 문서는 팀원들이 로컬 환경에서 각자의 인스타그램 계정을 연결하고 테스트하기 위한 설정 가이드입니다.

---

## 0. 사전 준비 (팀원 필수 수행) 🛠️

인스타그램 API를 사용하려면 **본인 소유의 페이스북 페이지**와 **인스타그램 비즈니스 계정**이 서로 연결되어 있어야 합니다. 아래 순서대로 세팅해 주세요.

### ① 인스타그램 계정을 '비즈니스'로 전환하기
API는 일반 개인 계정을 인식하지 못합니다. 반드시 '비즈니스 계정'이어야 합니다.
1. 인스타그램 앱 실행 -> 프로필 -> 우측 상단 [≡] 메뉴 -> **[설정 및 개인정보]**
2. **[계정 유형 및 도구]** -> **[프로페셔널 계정으로 전환]** 클릭
3. **[비즈니스]**를 선택하고 나머지 절차를 마칩니다.

### ② 페이스북 페이지 새로 만들기
1. 페이스북 로그인 후 왼쪽 메뉴의 **[페이지]** -> **[+ 새 페이지 만들기]** 클릭
2. 이름(예: `테스트_홍길동`)과 카테고리를 입력하고 생성합니다.

### ③ 페이지와 인스타그램 연결하기 (핵심!) 🔗
1. 새로 만든 **페이스북 페이지**의 설정 화면으로 들어갑니다.
2. **[설정]** -> **[연결된 계정]** -> **[Instagram]** 항목 선택
3. **[계정 연결]** 버튼을 누르고 본인의 인스타그램 계정으로 로그인하여 연결을 완료합니다.

---

## 1. Meta 앱 테스터 등록 과정 (매우 중요!) 👑

메타의 보안 정책상, 사장님이 팀원을 추가하려면 아래 과정을 반드시 거쳐야 합니다.

### ① 팀원: 페이스북 개발자 등록
팀원이 먼저 개발자로 등록되어 있어야 테스터로 추가할 수 있습니다.
1. 팀원이 [Meta for Developers](https://developers.facebook.com/) 접속 및 로그인
2. 우측 상단 **[시작하기]** 클릭 후 인증 절차를 거쳐 개발자 계정 생성 완료

### ② 사장님: 테스터 추가 (이름 또는 ID 사용)
이제 이메일 검색은 지원되지 않습니다.
1. 사장님은 [Meta 개발자 센터] -> [앱 역할] -> [역할] -> [테스터 추가] 클릭
2. 팀원의 **페이스북 실제 성함** 또는 **프로필 사용자 ID**를 입력하여 검색 후 추가

### ③ 팀원: 초대 수락
1. 팀원은 [초대 수락 페이지](https://developers.facebook.com/requests/)에 접속합니다.
2. 사장님이 보낸 앱 테스터 초대 요청을 찾아서 **[확인/수락]** 버튼을 누릅니다.

---

## 2. 프로젝트 로컬 환경 설정 (팀원 수행) 💻

1. GitHub에서 최신 코드를 pull 받습니다 (`refactor/song/main` 브랜치).
2. `.env.example`을 복사해 `.env` 파일을 생성합니다.
3. 사장님에게 공유받은 `META_APP_SECRET`과 `TOKEN_ENCRYPTION_KEY`를 넣습니다.
4. `META_REDIRECT_URI=http://localhost:8501/` 확인

---

## 3. 테스트 및 연결 진행 🚀

1. `streamlit run app.py` 실행
2. 사이드바의 **[인스타그램 연결하기]** 클릭 -> 페이스북 로그인
3. 로그인 팝업에서 **방금 만든 페이지와 인스타그램 계정**을 반드시 모두 체크하고 완료합니다.
