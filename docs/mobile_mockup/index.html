📱 모바일 앱 시뮬레이션

현재 app.py(Streamlit, 센터 레이아웃)의 기능을 모바일 앱 형태로 재구성한 정적 프로토타입이에요. 실제 코드는 수정하지 않았고, 레이아웃 검토용입니다.

매핑된 기능
  • 홈 — 온보딩 + 빠른 진입
  • 만들기 — tab_create 흐름 (상품 선택 → 목적 → 톤/스타일 → 생성)
  • 결과 — 텍스트/이미지 생성 결과 + 인스타 미리보기
  • 보관함 — tab_archive(예전에 만든 홍보물)
  • 설정 — render_sidebar_settings (Mock/실제 토글 등)

기술 메모
Streamlit 자체로 이 모양을 내려면 streamlit-option-menu로 하단 GNB, CSS로 사이드바/헤더 숨김, PWA 매니페스트 추가가 필요합니다.
